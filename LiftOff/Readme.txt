Design Thinking and GAE Tutorial for Orbital
-------------------------------------------

 * DesignThinking.pptx contains the slides for the Design Thinking tutorial
 * AnnexB.pdf contains example requirements for SMEs from IDA (described in Design Thinking tutorial)
 * GAE.pptx contains the slides for the GAE Tutorial
 * Mockup.pptx contains the powerpoint mockup of the app built in the GAE tutorial
 * first is the directory used in the first part of the tutorial
 * Giftbook is the directory used in the second part of the tutorial
 * GiftbookRec contains the a more complex version of the app with recommendation
